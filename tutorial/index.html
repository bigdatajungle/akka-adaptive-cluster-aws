Introduction

A seed 'one action' microservice using akka-http, akka cluster with adaptive-group strategy along with link to instructions how to install, configure and run the cluster on AWS instances.

Features

  • Akka HTTP implicit JSON marshalling: Json4sSupport.scala
  • Cake pattern for unit testing of the Akka router: FrontendSpec.scala
  • Become/Unbecome pattern for var-less actor: SampleClusterBackend.scala
